<Info>
  **SI-7 · Signal Intelligence · 300 XP · ~2 weeks**
</Info>

This capstone is different from the others. You're not just building something — you're running it. The measure of a signal workflow isn't whether it's set up correctly. It's whether it generates replies.

You'll run a live signal workflow for 2 weeks, document everything, and submit results for review.

---

## What You're Building and Running

A live signal-triggered outbound workflow covering:

1. **Signal Feed setup** — monitoring 3+ signal types for a defined ICP
2. **Signal scoring system** — composite signal × fit scoring
3. **Enrichment trigger** — automatic contact enrichment when signal fires
4. **Personalized copy generation** — signal-aware email for each triggered contact
5. **Live sends** — at least 30 signal-triggered emails sent via your sequencer
6. **Results tracking** — replies, sentiment, meeting booked rate
7. **Performance debrief** — what worked, what didn't, what you'd change

---

## 2-Week Timeline

**Week 1: Build and seed**
- Days 1–2: Set up Signal Feed grid and monitoring sources
- Days 3–4: Seed with first batch of signals, run scoring and enrichment
- Days 5–7: Generate copy, QA, send first 15–20 signal-triggered emails

**Week 2: Run and document**
- Days 8–10: Monitor replies, continue signal detection and sending
- Days 11–12: Complete 30+ total sends, track all results
- Days 13–14: Write performance debrief

## Performance Benchmarks

Your submission will be evaluated against these benchmarks:

| Metric | Minimum to Pass | Target |
|---|---|---|
| Signal types monitored | 3 | 4–5 |
| Signals detected | 20+ | 40+ |
| Emails sent | 30 | 50+ |
| Reply rate | > 5% | > 10% |
| Positive reply rate | > 2% | > 5% |
| Signal quality score documentation | Required | Required |

---

## Submission Requirements

1. Signal Feed grid (all rows, all columns, including enriched contacts)
2. Evidence of sends (screenshot from sequencer or export)
3. Reply tracking sheet (reply, sentiment, outcome)
4. 2-page performance debrief:
   - Which signal type generated the best results?
   - What signal quality score cutoff worked best?
   - Which email template had the highest reply rate?
   - What would you change in Week 3?
5. CSV export of full dataset
