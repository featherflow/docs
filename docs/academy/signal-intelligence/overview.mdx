# Signal Intelligence Track

Data tells you who to reach. Signals tell you when. This track is about the when.

Most outbound operates on an arbitrary schedule: "send 100 emails per day, run for 7 days." Signal intelligence inverts this: you reach out when something meaningful has happened, to the people most likely to care about it.

The result is outreach that feels timely and relevant — because it is.
