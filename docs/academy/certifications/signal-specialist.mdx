# Signal Intelligence Specialist

This certification demonstrates that you can build and operate signal-triggered GTM workflows — detecting the right moments, enriching the right contacts, and reaching out with outreach that feels timely and relevant.

## What This Certification Covers

- Signal taxonomy and quality scoring
- Job change detection and champion tracking
- Funding and growth signal interpretation and activation
- Hiring pattern analysis as an ICP qualification and timing signal
- News and event monitoring for outreach triggers
- Unified signal workflow architecture in Bitscale
- Live signal-triggered outbound campaign execution

## How to Earn It

Complete all modules in the Signal Intelligence track (SI-1 through SI-6) and pass the SI-7 capstone — which requires running a live workflow with verified send and reply data.

## Unique Requirement

Unlike other certifications, the SI-7 capstone requires a live run — you must submit actual send and reply data. Simulated data is not accepted. This ensures the certification reflects real operational experience.

  **Review timeline:** Capstone submissions are reviewed within 3 business days. Reply rate must be ≥ 5% to pass.
</Info>
