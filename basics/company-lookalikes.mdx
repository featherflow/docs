## Why lookalikes are a powerful list-building shortcut

One of the fastest ways to build a high-quality target list is to start with companies you already know.

If you have a partner account, a customer you delivered value to, or a company where your playbook already worked, you can use it to find similar companies.

This helps because:
- you already know the industry and problem well
- you already have proof that your approach works
- the next company is more likely to take your message seriously

## Two ways to find lookalikes in Bitscale

1. **Bulk lookalikes from one company**
   - best when you have one “perfect” account and want a larger list of similar companies
2. **Lookalikes for each company in a list**
   - best when you have multiple seed accounts and want a small set of lookalikes for each one

## Method 1: Bulk lookalikes from one company

Use this when you want to find a full list of similar companies from one strong seed account.

1. Click **New grid**
2. Select **Find company lookalikes**

<img
  style={{ maxWidth: "100%", borderRadius: "8px" }}
  src="https://res.cloudinary.com/dbfzpmxn2/image/upload/v1773219230/Screenshot_2026-03-11_140523_q1ptvw.png"
  alt="Selecting the Find company lookalikes option when creating a new grid"
/>

3. Paste the **company URL**
4. Add filters if needed
5. Choose how many results you want
6. Click **Import to new grid**

/>

Now you have a grid with lookalike companies, and you can:
- enrich them
- push them into your CRM as target accounts
- run outbound workflows or cadences

## Method 2: Find lookalikes for each company in a list

Use this when you already have a list of companies and want a small number of lookalikes for each one (like 5 to 15).

### Step 1: Add the lookalikes enrichment inside the grid

1. Open your grid
2. Click **Enrichments**
3. Search for **Lookalikes**
4. Select it to open the setup

<img
  style={{ maxWidth: "100%", borderRadius: "8px" }}
  src="https://res.cloudinary.com/dbfzpmxn2/image/upload/v1773219230/Screenshot_2026-03-11_140732_umndq4.png"
  alt="Opening the Lookalikes enrichment from the Enrichments menu inside a grid"
/>

### Step 2: Configure the enrichment

1. For **Company URL**, pass the website/domain column
2. Add the desired filters
3. Set the number of lookalikes per company (preferably 5 - 15)
4. **Save**
5. Run it for a test batch of **first 10 rows**

Now each row will return a set of lookalikes.

### Step 3: Export lookalikes into a new grid

To use the returned companies as a real list, export them into a separate grid.

1. Click a lookalikes cell
2. Choose **Export to grid**

<img
  style={{ maxWidth: "100%", borderRadius: "8px" }}
  src="https://res.cloudinary.com/dbfzpmxn2/image/upload/v1773219230/Screenshot_2026-03-11_141348_yq9vas.png"
  alt="Lookalike results in a grid with the Export to another grid option"
/>

3. Create a new grid
4. Include the original website URL along with the lookalike results
5. Save
6. Run it for the same rows you tested

When you open the new grid, the exported lookalike companies will appear in the source, ready to enrich and work.

## What to do next

Once your lookalike companies are in a grid, the most common next steps are:
- enrich company details
- find relevant people at those companies
- push to your CRM as target accounts
- trigger outbound workflows
