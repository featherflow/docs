    How to Create Action Columns on Our Platform

    In this walkthrough, we demonstrate selecting and configuring tools for your specific needs. Start with input columns like names and companies to create a one-liner outreach message.

    1. Create Action Column: Click to configure your tools.
    2. Select Tool: Choose from various categories. For text generation, select a model like GPT-4 mini.
    3. Configure Prompt: Enter your prompt using references (e.g., @name, @company) for personalization.
    4. Save and Run: Save your configuration and run the action column. You can process multiple rows at once.

    Explore the platform, and don’t hesitate to reach out for support!

    Discover Company Tech Stacks with BuildFit

    In this demo, learn how to identify the technologies used by any company using BuildFit.

    1. Set Up Your Grid: Start with a grid containing your leads and their website domains.
    2. Add Text Tech Column: Create a new column and select BuildFit as your tech stack partner.
    3. Select Domain Column: Choose the column that contains the company's website.
    4. Run the Tool: Save your configuration and run the tool on your selected rows.

    The results will reveal the technologies being used—over 160 tech stacks in some cases! Use this valuable data for further analysis or integration into your workflows.

    Import Leads from Sales Navigator

    We're excited to introduce a new feature that allows you to import leads directly into BitSkill using your Sales Navigator URL. Here's how it works:

    1. Open Sales Navigator: Apply your desired filters to find potential leads.
    2. Copy the URL: Once your filters are set, copy the Sales Navigator link from the top.
    3. Import into BitSkill: In the BitSkill tab, choose either "Companies" or "People," paste the URL, and specify the number of results (up to 1,000).
    4. Wait for Processing: After clicking "Import," wait 5 to 10 minutes for the system to process and enrich your grid.
    5. View Your Leads: Once completed, you can access all imported leads, complete with LinkedIn profiles and enriched data.
    
    Start streamlining your lead management process today!
